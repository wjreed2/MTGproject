Scanner pHash parity harness

Loads each Scryfall card image in the browser, computes the client pHash with js/phash-core.js (canvas resampling), and asks POST /api/scan/identify to match it against the server fingerprint DB (sharp resampling). A self-match should return distance 0 (perfect parity) — small distances (≤ a few) are acceptable resampling drift. The fingerprint DB must contain these printings (run npm run build:fingerprints first; the defaults below are in the smoke-test set).
